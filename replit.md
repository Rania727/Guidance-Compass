# بوصلة - BOSELAH

## Overview

Boselah (بوصلة) is a multilingual (Arabic/English, RTL/LTR) academic and career guidance platform. It helps students from high school to university make informed academic and career decisions through an interactive assessment that analyzes their interests, abilities, skills, and career preferences. The platform uses AI to generate personalized recommendations for university majors, career paths, and skills to develop — all tailored to the student's selected country and language.

Key features:
- **Language selection** as first step of assessment (Arabic/English with auto RTL/LTR switching)
- **Country selection** (25+ countries, defaults to Jordan) for localized job market recommendations
- **"Other" option** with free-text input on every question
- **Bilingual questions** served from backend with `text`/`textEn` fields
- **AI prompts** generated in the selected language, referencing the student's country for realistic salary ranges, demand levels, and universities
- **Transparency section** on home page (no fake stats), explaining how AI works and data privacy
- **Contact info** (email + phone) in footer and results page
- **Expanded career preferences** with 6-7 options per question covering remote work, entrepreneurship, direct employment, and study abroad

pnpm workspace monorepo using TypeScript. Each package manages its own dependencies.

## Stack

- **Monorepo tool**: pnpm workspaces
- **Node.js version**: 24
- **Package manager**: pnpm
- **TypeScript version**: 5.9
- **API framework**: Express 5
- **Database**: PostgreSQL + Drizzle ORM
- **Validation**: Zod (`zod/v4`), `drizzle-zod`
- **API codegen**: Orval (from OpenAPI spec)
- **Build**: esbuild (CJS bundle)
- **Frontend**: React + Vite + Tailwind CSS + Shadcn/UI
- **AI**: OpenAI via Replit AI Integrations (gpt-5.2)
- **Animation**: Framer Motion

## Key Features

1. **Landing Page** - Arabic RTL hero section with platform branding
2. **Multi-step Assessment** - 4 categories of questions (interests, academics, skills, career preferences)
3. **AI-Powered Results** - Personalized recommendations generated by GPT-5.2
4. **Results Page** - Personality profile, recommended majors with match %, career paths, skill recommendations

## Database Schema

- **assessment_results** - Stores student assessments with AI-generated recommendations (JSONB)
- **majors** - University majors with descriptions, skills, duration
- **careers** - Career paths with salary info, demand level, required majors

## API Endpoints

- `GET /api/assessment/questions` - Returns assessment questions grouped by category
- `POST /api/assessment/submit` - Submits answers and generates AI recommendations
- `GET /api/assessment/results/:id` - Gets saved assessment result
- `GET /api/majors` - Lists all university majors
- `GET /api/careers` - Lists all career paths
- `GET /api/healthz` - Health check

## TypeScript & Composite Projects

Every package extends `tsconfig.base.json` which sets `composite: true`. The root `tsconfig.json` lists all packages as project references.

## Root Scripts

- `pnpm run build` — runs `typecheck` first, then recursively runs `build` in all packages
- `pnpm run typecheck` — runs `tsc --build --emitDeclarationOnly` using project references
- `pnpm --filter @workspace/scripts run seed` — seeds the database with majors and careers data

## Packages

### `artifacts/boselah` (`@workspace/boselah`)

React + Vite frontend for the Boselah platform. Fully Arabic with RTL layout support.

### `artifacts/api-server` (`@workspace/api-server`)

Express 5 API server with assessment, majors, and careers routes. Uses OpenAI for AI-powered recommendations.

### `lib/db` (`@workspace/db`)

Database layer with Drizzle ORM. Tables: assessments, majors, careers.

### `lib/integrations-openai-ai-server`

OpenAI integration via Replit AI Integrations proxy. No API key needed.
